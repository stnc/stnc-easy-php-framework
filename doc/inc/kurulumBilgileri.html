Kurulum gereklilikleri

Framework'ün kurulum gereksinimleri çok azdır

  • Apache Web Server veya mod rewrite modunda yazılabilen bir sunucu (lightwiev,nginx)
  • Windows için kullaılaksa IIS için URL Rewrite modülü yüklenmelidir http://www.iis.net/downloads/microsoft/url-rewrite
  • PHP 5.3 ve sonrası ile çalışır

Bir veritabanı gerekli olmasa da, sistem MySQL veritabanı ile çalışmak için tasarlanmıştır.
Çok gerekli görülürse model yapısı değiştirilebilir.
