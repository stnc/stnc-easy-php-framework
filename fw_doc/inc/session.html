Session Sınıfı

Session, bir oturum nesnesidir. Cookie den daha güvenli olarak bilgi saklamaya ,bilgiye tüm sayfalardan erişmeye yarar


session nesnesi oluşturmak

			
        // ilk olarak use dan tanıtalım 
        //use \Lib\Session;
  session::set('message', 'merhaba');
//message isminde bir session oluşturduk

session nesnesini okumak

			
  echo session::get('message');
//message isminde bir session oluşturduk

session nesnesini silmek

			
  session::pull('message');

tüm session'ları okumak

			
    print_r(session::display());

tüm session'ları silmek

			
       session::destroy();
